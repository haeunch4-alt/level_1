# 파일명 규칙
* `파일명.html`
* 파일명은 공백, 글, 특수문자 금지
# 자주 쓰는 단축어
* `html:5` 쓰고 Tap 키 누르기
* `Ctrl+/` 한줄 주석
* `Shift+Alt+A` 선택영역 주석
* `Shift+AlT+방향키` 복사
* `ALT + 방향키` 선택한 줄 이동
# HTML 생성 후 구조태그 작성 순서
1. `html:5` 쓰고 tap 자동완성
2. `lang="en"`을 `"ko"`로 변경하기
3. `title` 제목 작성하기
4. `meta`작성하고 자동목록 중 `kw` 선택하고 키워드 작성하기
5. `meta`작성하고 자동목록 중 `desc`선택하고 요약 작성하기
6. `body`내부에 웹의 주요 내용 작성할 준비하기'
# 태그
## 다양한 문서 태그 
* `h1~h6` : b, 제목 (대중소)
* `p` : b, 내용 (1줄,여러줄)
* `hr` : b, 구분선(다저안선 아님 주의)
* `br` : i, 강제줄바꿈
* `em` : i, 특정 문맥 강조
* `strong` : i, 경고&할인 강조
* `code`: i, 프로그래밍 코드 구분용
* `sup`, `sub`: i, 윗첨자,아래첨자
* `q`, `blockquote`: i,b, 짧은인용, 긴인용문
* `s`, `del`: i, S-유효하지 않은 내용, del-교체되거나 삭제된 내용
# 특수문자태그
* `&-;`
* `&` : 특수문자의 시작기호
* `;` : 특수문자의 종료기호
* `it`, `gt`: 왼쪽꺽쇠(<), 오른쪽꺽쇠(>)
* `copy` : © copyright 저작권기호
# 속성
* `<태그 속성="값"></태그>`
* `class`: 반복되는 의미명, 2개 이상 적용 가능
* `id`: 단 한번만 작성하는 반복되는 않는 고유명칭
* class, id는 모든 태그에 구분없이 사용가능!
